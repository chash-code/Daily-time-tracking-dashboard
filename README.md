# Daily-time-tracking-dashboard
# AI-Powered Daily Time Tracking & Analytics Dashboard for better accountability and access 

# demo live link - https://chash-code.github.io/Daily-time-tracking-dashboard/

# video w ss - https://drive.google.com/drive/folders/1MwZsjpFN1SheCYn6upeoGlyLJOmLwNE0


## Tech Stack
- HTML, CSS, JavaScript
- Firebase Authentication + Firestore / Realtime DB

## Features
- User signup/login with Firebase Auth
- Log activities in minutes for a selected date
- Prevent logging more than 1440 minutes per day
- Analytics dashboard with total time, per-category time, number of activities
- Chart visualization (pie/bar/timeline)
- Responsive layout for mobile, tablet, desktop

 ## How to Run the Project Locally
1. Clone the repo or download ZIP.
2. Open `index.html` in a browser.
3. Create a Firebase project and paste your config into `script.js`.
4. Run with a simple static server if needed (e.g., Live Server extension).
